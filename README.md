## 요구사항

### 기본
#### 인증
- [x] `User` 스키마를 작성하기
- [x] 회원가입 API를 만들기
- [x] 토큰 기반 인증: 로그인에 성공하면 Access Token을 발급하는 기능을 구현하기 
#### 상품 기능 인가
- [x] 로그인한 유저만 상품을 등록 가능
- [x] 상품을 등록한 유저만 해당 상품의 정보를 수정하거나 삭제 가능
#### 게시글 기능 인가
- [x]  로그인한 유저만 게시글을 등록 가능
- [x] 게시글을 등록한 유저만 해당 게시글을 수정하거나 삭제 가능
#### 댓글 기능 인가
- [x] 로그인한 유저만 상품과 게시글에 댓글을 등록 가능
- [x] 댓글을 등록한 유저만 해당 댓글을 수정하거나 삭제 가능
#### 유저 정보
- [x] 자신의 정보를 조회, 수정하기
- [x] 자신이 등록한 상품 목록을 조회하기

### 심화
#### 인증
- [x] 토큰 기반 인증: Refresh Token으로 토큰을 갱신
#### 좋아요 기능
- [x] 로그인한 유저는 상품과 게시글에 '좋아요'와 '좋아요 취소'를 할 수 있음
- [x] 상품 또는 게시글을 조회할 때 유저가 '좋아요'를 누른 항목인지 확인할 수 있도록 `isLiked`라는 불린형 필드를 포함하여 리스폰스함
- [x] 유저가 '좋아요' 표시한 상품의 목록을 조회
