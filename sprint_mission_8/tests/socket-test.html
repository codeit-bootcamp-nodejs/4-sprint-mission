📝 테스트 방법

  1. POST /auth/login으로 로그인하여 access-token 쿠키 값을 복사합니다
  2. 위 입력창에 토큰을 붙여넣고 "연결" 버튼을 클릭합니다
  3. "브라우저 알림 허용" 버튼을 눌러 데스크톱 알림을 활성화합니다 (선택)
  4. 다른 브라우저/탭에서 다음 작업을 수행합니다:
    • 다른 사용자가 내 게시글에 댓글 작성
    • 다른 사용자가 내 상품에 댓글 작성
    • 내가 찜한 상품의 가격 변경
  5. 이 페이지에서 실시간으로 알림이 표시되는지 확인합니다!
